# NeuralRetail – AI Sales Intelligence & Predictive Analytics

## Overview

NeuralRetail is an end-to-end retail analytics project focused on:

- Sales Intelligence
- Customer Segmentation
- Demand Forecasting
- Churn Prediction
- Inventory Optimization

This project combines:

- Python
- Machine Learning
- Power BI
- Data Analytics

to deliver business insights for retail and e-commerce operations.

---

## Tech Stack

- Python
- Pandas
- Scikit-learn
- XGBoost
- Prophet
- Power BI
- Plotly

---

## Features

- Executive KPI Dashboard
- Sales Trend Analysis
- Customer Analytics
- Forecasting Models
- Inventory Insights
- Interactive Visualizations

---

## Project Structure

- data/ → raw and processed datasets
- notebooks/ → analysis notebooks
- scripts/ → reusable Python scripts
- powerbi/ → dashboard files
- reports/ → final reports
- visuals/ → screenshots and charts
